### Specific Form Variations

#### 8. RESERVATIONS FORM (`/reservations`)
- **Fields:** Name, Phone, Email, Dining Space, Date, Time, Party Size, Occasion, Special Requests
- **Hero:** "Reserve Your Table at Amante"
- **Submit:** "Reserve My Table"

#### 9. PRIVATE EVENTS FORM (`/private-events`)
- **Fields:** Name, Phone, Email, Company, Event Type, Date, Alternate Date, Guest Count, Space Preference, Budget Range, Requirements
- **Hero:** "Plan Your Private Event"
- **Submit:** "Request Event Proposal"

#### 10. BANQUETS FORM (`/banquets`)
- **Fields:** Name, Phone, Email, Event Type, Event Date, Alternate Date, Attendance, Preferred Hall, Meal Type, Menu Preferences (checkboxes), Budget, Additional Requirements
- **Hero:** "Book Our Grand Banquet Halls"
- **Submit:** "Request Banquet Proposal"

#### 11. CONTACT FORM (`/contact`)
- **Fields:** Name, Email, Phone (optional), Subject (dropdown), Message
- **Hero:** "Get in Touch"
- **Submit:** "Send Message"

#### 12. FEEDBACK FORM (`/feedback`)
- **Fields:** Name (optional), Email (optional), Visit Date, Space Visited, Overall Rating (stars), Food/Service/Ambiance/Value (stars), What You Loved, What to Improve, Would Recommend (radio), Additional Comments
- **Hero:** "Share Your Feedback"
- **Submit:** "Submit Feedback"

#### 13. CAREERS FORM (`/careers`)
- **Fields:** Full Name, Email, Phone, Position Applying For, Current Location, Experience (years), Current Position, Expected Salary, Resume Upload, Portfolio Link, Why Amante (textarea), Available to Join (date)
- **Hero:** "Join Our Team"
- **Submit:** "Submit Application"

## MOBILE FORM ADAPTATIONS

All forms adapt for mobile with:
- Single column layout
- Full-width inputs
- Larger touch targets (48px minimum)
- Native date/time pickers
- Sticky submit button (optional)
- Simplified multi-column fields to stacked

## RESPONSIVE BEHAVIOR SUMMARY

### Desktop (1024px+)
- Two-column layouts for forms
- Side-by-side content and images
- Grid layouts: 3 columns for cards
- Sticky navigation and CTAs
- Hover effects active
- Larger typography

### Tablet (768px - 1023px)
- Two-column grids
- Slightly reduced typography
- Maintained hover effects
- Adapted navigation (may condense)

### Mobile (< 768px)
- Single column layouts
- Stacked cards and content
- Hamburger menu navigation
- Simplified typography
- Touch-optimized interactions
- Bottom navigation bars for key actions
- Reduced imagery (smaller hero heights)

## WIREFRAME NOTES FOR DEVELOPERS

### Layout Grid
- Use CSS Grid or Flexbox
- 12-column grid system
- Gap: 24px (desktop), 16px (mobile)
- Max container width: 1400px

### Images
- All images should use Next.js Image component
- Lazy load below-the-fold images
- WebP format with JPG fallback
- Responsive srcsets

### Forms
- Use React Hook Form
- Client-side validation with Zod
- Show validation on blur, not on every keystroke
- Success states replace form entirely
- Error messages appear below fields

### Interactive Elements
- All clickable elements should have cursor: pointer
- Loading states for async actions
- Disabled states visually distinct
- Focus states always visible

### Accessibility
- Semantic HTML (header, nav, main, section, footer)
- Proper heading hierarchy (h1-h6)
- Alt text for all images
- ARIA labels for icon-only buttons
- Focus trap in modals
- Keyboard navigation support
